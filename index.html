Key Technology

Pneumatic Alignment System

Inspired by PAS Tech used in competitive VEX IQ — our pneumatic system precisely extends and retracts the dusting head, letting the robot align to walls and apply consistent pressure for maximum dust removal.

💨

Extend

Pneumatics push the duster outward to contact the surface.

🎯

Align

System self-corrects to stay parallel to walls.

🔁

Retract

Mechanism pulls back when turning or navigating obstacles.

Robot Close-Up

Inside the Build

  • VEX IQ Brain visible at center — the robot's "brain"
  • Multiple smart motors connected via black cables
  • Pneumatic reservoir and cylinder in the base
  • Blue gear (large ratio) for torque on the lift
  • Mecanum / omni wheels for maneuverability
  • Clean cable management throughout
Programming

VEX IQ Block Code

  • Built in VEX IQ block code (IQ2 Brain)
  • Controller axes A, C, D mapped to drive motors
  • Left Drive & Right Drive motor groups
  • Pneumatic cylinder triggered by button events
  • Auto-calibration sequence on startup
  • Degree-based sensor tracking for position
